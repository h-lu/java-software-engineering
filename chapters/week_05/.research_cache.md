## AI 小专栏 #2: AI 生成集合操作代码的审查要点

### 搜索词: "AI code review Java collections type safety 2026"
- **关键最佳实践**（2025年更新）:
  1. 始终使用泛型，避免原始类型
  2. 使用 `isEmpty()` 而非 `size() == 0`
  3. 返回空集合而非 null
  4. 使用 `entrySet()` 遍历 Map 更高效
- 来源: https://www.javaguides.net/2025/02/top-15-java-collections-and-generics.html
- 来源: https://www.javaguides.net/2024/05/java-collections-best-practices-for-developers.html

### 搜索词: "Java collections common bugs AI generated code"
- AI 生成代码中常见的集合类问题：
  - 忘记检查空集合就遍历
  - 在遍历时修改集合导致 ConcurrentModificationException
  - 使用原始类型失去类型安全
  - 选择错误的集合类型（如用 ArrayList 做频繁查找）
