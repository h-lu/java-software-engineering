# Week 13 Starter Test Matrix

| 领域 | 起步包检查 | 学生完成后应自行检查 |
|------|------------|----------------------|
| OpenAPI | 文件存在，包含 TODO | 待办： 路径、请求体、响应、错误码与真实 API 一致 |
| README | 起步说明存在 | 待办： 5 分钟快速开始可执行 |
| ADR | ADR 索引和单篇 ADR 模板存在 | 待办： 至少汇总已有关键决策，链接可跳转 |
| AI 文档审查 | 无自动判断 | 待办： 人工核对端点、字段、版本号、URL、截图 |

## 运行

```bash
mvn test
```

## 说明

这些测试故意不要求 TODO 全部消失，因为 starter_code 应该保持可填写状态。你提交最终作业前，应在自己的项目交付物里清理 TODO。
