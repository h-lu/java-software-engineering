# Week 13 Starter Test Cases

本文件描述起步包自带的 smoke tests。它们只验证模板是否存在，不验证你的最终作业是否优秀。

| 测试类 | 测试内容 | 为什么保留 |
|--------|----------|------------|
| `Week13StarterSmokeTest` | `README.md` 包含运行命令和主要模板路径 | 防止 starter 包缺少说明 |
| `Week13StarterSmokeTest` | `openapi.yaml`、`PROJECT_README_TODO.md`、`docs/ADR_INDEX.md`、ADR 模板仍包含 TODO | 确认这是学生可填写的骨架 |

## 完成作业后建议新增的测试

- TODO: OpenAPI 文件是否包含你的真实端点。
- TODO: README 中的命令是否能在干净环境中运行。
- TODO: ADR 索引中的链接是否存在。
- TODO: 文档中是否没有敏感信息、编造链接或无效部署地址。
