# Week 16 Starter Code：项目展示与工程复盘

Week 16 的主要交付物是展示反馈、工程复盘、知识地图和职业规划。本 starter_code 是模板与检查器，不包含完整答案。

## 运行命令

需要 Java 21 和 Maven 3.9+。

```bash
cd chapters/week_16/starter_code
mvn test
```

可选：编译后运行检查器，查看模板中剩余 TODO 数量。

```bash
mvn -q compile
java -cp target/classes com.campusflow.Week16ReadinessCheck
```

## 需要编辑的文件

- `01_showcase_feedback_summary.md`：TODO 汇总项目集市反馈，可作为 `01_展示/反馈汇总.md`，也可以导出为 PDF。
- `02_retrospective_report.md`：TODO 使用 KPT 写工程复盘，最终导出为 `02_复盘/工程复盘报告.pdf`；能力雷达图需另存为 PDF 或图片。
- `03_knowledge_map.md`：TODO 画出 16 周知识地图和核心概念演进，最终导出知识地图 PDF/图片，并整理课程总结 PDF。
- `04_career_plan.md`：TODO 写职业路线、6 个月计划和技能差距，最终导出职业规划报告和学习计划 PDF/表格。
- `submission_checklist.md`：TODO 对照最终压缩包结构逐项检查。

## 注意事项

- 复盘必须基于你自己的真实经历，不要把示例改几个词就提交。
- 能力评分要解释理由，尤其要说明 Week 01 与 Week 16 的变化。
- 如果使用 AI 润色，保留 AI 辅助说明，并人工核对所有事实。

## 最终提交物映射

Markdown 模板是源稿，不等同于最终 zip。提交前请按 `ASSIGNMENT.md` 整理为 `01_展示/`、`02_复盘/`、`03_知识地图/`、`04_职业规划/` 四个目录，并把需要展示的图表、PDF、图片或表格放到对应目录。
