# Week 01 作业：你的第一个 Python 程序

## 作业目标

完成本作业后，你将能够：

1. 编写并运行一个完整的 Python 程序
2. 使用变量存储和展示信息
3. （可选）与用户进行简单的交互

---

## 任务说明

### 任务 1：Hello World 变体

**目标**：运行你的第一个 Python 程序，并学会修改它。

**步骤**：

1. 创建文件 `hello.py`
2. 编写代码，输出以下内容：

```text
Hello, Python!
我是 <你的姓名>
这是 Week 01 的作业
```

3. 运行程序验证输出：

```bash
python3 hello.py
```

**提示**：
- 使用 `print()` 函数输出每一行
- 将 `<你的姓名>` 替换为你的真实姓名

---

### 任务 2：个人信息卡片

**目标**：使用变量存储信息，然后格式化输出。

**步骤**：

1. 创建文件 `profile.py`
2. 定义以下变量：
   - `name`：你的姓名（字符串）
   - `age`：你的年龄（整数）
   - `city`：你所在的城市（字符串）
   - `hobby`：你的爱好（字符串）
3. 使用这些变量输出一张格式化的个人信息卡片：

```text
========== 个人信息卡片 ==========
姓名: <name>
年龄: <age>
城市: <city>
爱好: <hobby>
==================================
```

**要求**：
- 必须使用变量存储信息
- 输出格式必须与示例一致（包括分隔线）

---

### 任务 3（可选）：简单对话程序

**目标**：读取用户输入并做出响应。

**步骤**：

1. 创建文件 `chatbot.py`
2. 实现以下交互流程：

```text
你好！我是 Python 小助手。
请问你叫什么名字？> 张三
很高兴认识你，张三！
你今年几岁了？> 20
哇，20 岁，正是学习编程的好年纪！
```

**提示**：
- 使用 `input()` 函数获取用户输入
- 使用 f-string 将用户输入嵌入到回复中

| 任务 3（可选） | `chatbot.py` |

### 代码规范

1. **文件头注释**：每个文件开头添加以下注释

```python
# 文件名: <文件名>
# 作者: <你的姓名>
# 日期: <YYYY-MM-DD>
# 描述: <简要描述这个程序做什么>
```

2. **变量命名**：使用有意义的英文单词或拼音，例如：
   - 好的：`name`, `user_age`, `city`
   - 避免：`a`, `b`, `x1`

3. **代码整洁**：
   - 运算符两边加空格：`name = "张三"`（不是 `name="张三"`）
   - 适当空行分隔逻辑块

### 提交方式

将所有 `.py` 文件打包提交到课程平台，或按教师要求的方式提交。

## 常见问题

**Q: 我的代码运行时报错 `SyntaxError`，怎么办？**

A: 检查以下几点：
- 所有字符串是否都用引号包裹（`"` 或 `'`）
- 括号是否成对出现
- 是否混用了中文标点和英文标点（Python 只认英文标点）

**Q: 任务 2 的年龄需要是数字，但输出时想和其他文字一起显示？**

A: 使用 f-string：

```python
age = 20
print(f"年龄: {age}")
```

**Q: 任务 3 的 `input()` 获取的是字符串，怎么转换成数字？**

A: 使用 `int()` 函数：

```python
age_str = input("你今年几岁了？> ")
age = int(age_str)
```
