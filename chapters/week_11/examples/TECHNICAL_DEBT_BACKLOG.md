## 优先级定义

| 优先级 | 说明 | 响应时间 |
|--------|------|---------|
| P0 | 严重问题：影响核心功能、安全风险 | 本周必须修复 |
| P1 | 重要问题：影响质量基线 | 本周完成 |
| P2 | 一般问题：优化类、改进类 | 下周修复 |
| P3 | 低优先级：锦上添花 | 有空时处理 |

## 类型定义

| 类型 | 说明 | 示例 |
|------|------|------|
| Bug | 明确的缺陷 | 空指针、资源泄漏 |
| Test | 测试覆盖率不足 | 缺少边界测试 |
| Clean | 代码清理 | 未使用的变量、导入 |
| Design | 设计问题 | compareTo 与 equals 不一致 |
| Perf | 性能问题 | 不必要的对象创建 |
